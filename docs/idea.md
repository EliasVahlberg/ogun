# Underlying idea

A lot of work has been done when it comes to algorithms and AI models for PCB design and semiconductor architecture which I belive can also be applied on larger (and more chaotic systems) such as town/city infrastructure where a lot of paralells can be drawn between traces, components to roads and buildings, the paralells.md document describes this further. What I want to do is to borrow algorithms and models meant for pcb design and use them for procedural town/city generation (and hopefully use it in my game saltglass-steppe).

Some of my inspiration comes from the SYNTHETIK 1 and 2.
